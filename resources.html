Useful Resources

There are many useful website that I have used to improve my webpage
Here are the ones that I consider as the most useful:

  • MDN Web-Docs
  • Mozilla web developers website is an equally great resource for both beginning and seasoned web developers. In includes information about the latest trends and developments as well as a huge base of guides and tutorials for newer users.
  • Unsplash
  • Unsplash is agreat resource for royalty free, high quality images that can be used as backgrounds or banners for your website.
  • Table of Html/Xml references
  • This table of HTML references came to my rescue a huge number of times. It's hosted on Wikipedia and consits of a huge table of HTML resources that proved very useful to me quite a few times.
  • Google Developers
  • Google developers is a great source of information about responsive design, accessibilty and other newest web technologies.
  • Subtle Patterns
  • Subtle patterns is another great image source. The website provides royalty free, subtle background images that stay consisten through the entire page without any tears or ugly borders.
  • Lynda.com
  • Lynda.com is the best internet resource when it comes to learning new things. It includes full classes and lessons on just about any topic you could imagine. It hugely helped me with understanding the basics of HTML and CSS
